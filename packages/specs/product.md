# Product Overview

## Summary

A multi-tenant SaaS product with authenticated dashboard access and CRUD management for projects.

## Roles

- Owner: manages organization, members, billing, projects
- Admin: manages members and projects
- Member: accesses projects based on membership

## Key Flows

1. User signs in.
2. User selects active organization/tenant.
3. User accesses dashboard.
4. User performs project CRUD actions.
5. Organization billing state gates premium capabilities.
